# MyChoice Alpha — Field-Test Documents

This folder contains documents used with families participating in the **MyChoice Alpha field test**.

| File | Audience | Purpose |
|---|---|---|
| `parent-data-download-guide.html` | Participating parents/guardians | Friendly walkthrough for requesting and preparing platform data exports |
| `alpha-data-consent-and-confidentiality.md` | Participating families + MyChoice team | Plain-English consent and confidentiality terms for alpha data handling, signed alongside our standard NDA |

## Status and limitations

- These are **alpha field-test documents**, written in plain English for a small, known group of participants.
- They are **not final legal documents**. They have not been reviewed by counsel.
- **Before any broader use** (more participants, public beta, any minor-participant expansion, or use outside the US), these documents must be reviewed by qualified counsel — in particular for COPPA (under-13), state teen-privacy laws, and GDPR/GDPR-K if any participant is in the EU/UK.
- The consent document deliberately describes the alpha system's security posture honestly (local/controlled-environment processing, not production-grade). Do not edit it to claim stronger guarantees than the system actually provides.
- The standard company NDA covers business confidentiality only; it is **not sufficient** on its own for family-data consent, which is why the consent document exists.

## Change control

Treat both documents as versioned artifacts: any change that affects what participants agreed to requires re-consent from affected families.
